## Rig-EternalAI

> **⚠️ UNMAINTAINED — This crate is no longer actively maintained and has been archived.**
>
> Eternal AI has pivoted away from text completions to image and video generation,
> making this integration non-functional. The code is preserved here for reference only.
>
> If Eternal AI reintroduces a completions API in the future, please open an issue
> in the main repository to discuss reviving this crate.

---

This companion crate integrated EternalAI and their blockchain capabilities with Rig.

## Usage (historical)

```toml
[dependencies]
rig-eternalai = "0.1.0"
rig-core = "0.4.0"
```

See the [`/examples`](./examples) folder for usage examples.
